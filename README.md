# Online-Banking-Simulator
A Java-based program that simulates an online banking system. This project includes a fully functional Admin Panel and core banking features such as:

- Depositing
- Withdrawing
- Transferring
- Password resets
- Logging in and out

If you have any questions, concerns, or advice to help improve this project, feel free to contact me!


Features
--------

- User Accounts – Create, log in, and manage personal banking accounts
- Creating Accounts – Creates a new account within the banking system
- New Account Number – Creates a new account for the currently logged-in user
- Admin Panel – View all users, reset passwords, and manage system data
- Secure Login System – Password-protected access with reset functionality
- Transaction System – Deposit, withdraw, and transfer funds between accounts
- Session Handling – Log in and out safely
